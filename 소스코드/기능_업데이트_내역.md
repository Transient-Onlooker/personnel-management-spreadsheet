# 인원관리 프로그램 기능 업데이트

## 2025-10-24: 인원 정보 편집 기능 추가

- **기능:** 설정 탭의 인원 목록에서 특정 인원의 정보를 수정할 수 있는 기능이 추가되었습니다.
- **사용법:**
    1.  `설정` 탭으로 이동합니다.
    2.  인원 목록에서 수정하고 싶은 인원을 더블클릭합니다.
    3.  '인원 정보 수정' 창이 나타나면 이름, 전화번호, 역할, 참여 시간을 수정한 후 '저장' 버튼을 클릭합니다.
- **참고:**
    -   수정된 정보는 Google Sheets와 자동으로 동기화됩니다.
    -   기존과 동일하게 인원 추가 및 삭제 기능도 사용할 수 있습니다.
